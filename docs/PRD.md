# Product Requirements Document (PRD)

## Project Overview
Build a classification model for the Iris dataset.

## Requirements

1. **Model Type**: Classification algorithm.
2. **Dataset**: Iris Dataset (includes features and response).
3. **Data Splitting**: 80% of data for training (to find weights $w$) and 20% for testing.
4. **Loss Function**: Mean Square Error (MSE).
5. **Outputs (Visual Metrics)**:
   - Generate a 4x4 Confusion Matrix.
   - Calculate Accuracy score.
   - Create a Convergence Graph showing the error decreasing over iterations.
6. **Final Output**: Produce a summary report including model description, data insights, and graphs.
